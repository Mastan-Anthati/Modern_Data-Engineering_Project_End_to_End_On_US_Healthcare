# Scripts

This folder contains ready-to-use SQL scripts for the healthcare project.

## Files
- create_healthcare_star_schema.sql: creates the star schema tables in Azure SQL Database.

## Usage
1. Open the SQL script in Azure Data Studio or SQL Server Management Studio.
2. Connect to your Azure SQL Database.
3. Run the script.
4. Use the generated tables as the target for your ADF/Databricks ingestion flow.
